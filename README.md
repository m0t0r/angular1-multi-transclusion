# angular1-multi-transclusion
A simple calendar component that uses angular 1.5 multiple transclusion feature

## Usage:

`npm install`

```html
<my-calendar>
  <day-of-week>Tuesday</day-of-week>
  <date>17</date>
  <month>October</month>
</my-calendar>
```
